# SWAWS Solutions Website

Static marketing website for **SWAWS Solutions**, a networking and infrastructure company.

## Tech Stack

- HTML
- CSS
- JavaScript (no frameworks)

## Structure

- `index.html` – Main page with hero, why choose us, business units, services/testimonials, value propositions, packages, news & insights, careers, and contact sections.
- `styles.css` – Global styles, layout, color system, and responsive behavior.
- `script.js` – Navigation behavior, smooth scrolling, mobile menu toggle, and simple contact form handling.

## Running the Site

No build step is required:

1. Open `index.html` directly in your browser, or
2. Serve the folder with a simple static server (for example, using VS Code Live Server or `npx serve`).

## Customization

- Update text content directly in `index.html`.
- Adjust colors, spacing, and typography in `styles.css`.
- Extend or replace the simple form handling logic in `script.js` if you connect to a backend or form service.
